  • - 비속어→순우리말 바꾼 예시
    1. 미친 → 정말 대단하다 / 정말 놀랍다
    - 예) "미친 일이야" → "정말 대단한 일이야"
    2. 씨발 → 정말
    - 예) "씨발, 어떻게 된 거야?" → "정말, 어떻게 된 거야?"
    3. 개새끼 → 못된 사람 / 나쁜 사람
    - 예) "그 개새끼가 뭐야?" → "그 못된 사람이 뭐야?"
    4. 좆같다 → 끔찍하다 / 형편없다
    - 예) "좆같은 상황이야" → "정말 형편없는 상황이야"
    5. 병신 → 무례한 사람 / 어리석은 사람
    - 예) "병신 같은 짓" → "어리석은 짓"
    6. 닥쳐 → 조용히 해 / 잠시 침묵해
    - 예) "닥쳐!" → "조용히 해!"
    7. 꺼져 → 나가세요 / 떠나세요
    - 예) "꺼져!" → "나가세요!"
    8. 지랄하다 → 엉뚱한 소리하다 / 괜한 짓하다
    - 예) "지랄하지 마!" → "엉뚱한 소리하지 마!"
